OUT-OF-THE-BOX-TEST


Methodenbeschreibung:

Nutzer werden in ihrem natürlichen Umfeld gefilmt, wie sie sich mit dem interaktiven System vertraut machen, dies beinhaltet das Auspacken, den Aufbauen, das Anfassen, den initialen Start und das Installieren möglicher Software. Dadurch erhält man Einblick in typische Nutzungsszenarien und Probleme mögliche Nutzer mit dem System haben könnten.


Ablauf/Anleitung:

Nutzer bekommen das interaktive System zunächst nach hause geschickt damit sie in ihrer natürlichen Umgebung sind und filmen sich um zu vermeiden, dass die Nutzer nervös sind wenn ein Testleiter ihnen bei der erstmaligen Benutzung eines neuen Systems zuschaut. Der Nutzer packt das System zunächst aus der Box und führt einen erstmaligen Set up durch. Bei der Analyse des Videomaterials wird darauf geachtet wie lange der Nutzer zum Beispiel für die einzelnen Schritte benötigt, an welchen Stellen er Fehler macht, oder sich unsicher ist. Das Ziel einer Out-of-the-Box Analyse ist es, das interaktive System und dessen Nutzung so intuitiv wie möglich zu gestalten. Nach der Analyse wird das System modifiziert um dieses Ziel bestmöglichst zu erreichen.

Ergebnisse/Was man mit dieser Methode erreichen will:

Mit Hilfe der Out-of-the-box Methode erhält man qualitatives Videomaterial, anhand dem man das interaktive System verbessert damit es so intuitiv wie möglich zu nutzen ist.
